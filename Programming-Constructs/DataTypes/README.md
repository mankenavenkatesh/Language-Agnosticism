# Data Types

Rust is
- Statically typed language
- The compiler can usually infer what type we want to use based on the value and how we use it
- 

## Data type subsets: 
- scalar 
-  compound

### Integer
An integer is a number without a fractional component. 
```
Length	Signed	Unsigned
8-bit	i8	    u8
16-bit	i16	    u16
32-bit	i32	    u32
64-bit	i64	    u64
128-bit	i128	u128
arch	isize	usize
```
